# swiggy-analysis
This project analyzes Swiggy sales data to identify customer behavior and city-wise performance.
## 📊 Tools Used
- Power BI
- Excel
- SQL

## 🔍 Key Insights
- Top cities contributed highest revenue
- Customer ordering patterns identified
- Peak sales time analyzed
